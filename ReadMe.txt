# 🎓 Student Marks Analyzer with Report Card PDF Generator

A Streamlit-based web application that allows teachers or schools to analyze student marks, visualize performance trends, generate professional report cards as PDFs, and optionally support multi-language and logo branding.

---

## 🚀 Features

- 📊 Upload or manually enter student marks
- 🎨 Beautifully styled visualizations (bar charts, heatmaps)
- 📄 Downloadable **report cards as PDF** (per student or all combined)
- 🏫 School name, address, and custom logo support
- 🏷️ Automated **grade assignment** (A+, A, B, C, D, F)
- 🌐 Language toggle: **English / Hindi** (can be extended)
- 📈 Subject-wise and student-wise performance visualization
- 🖼️ Logo upload for PDF branding
- 🔥 Seaborn heatmap for quick performance insights

---

## 🧰 Technologies Used

- **Python 3**
- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/)
- [Matplotlib](https://matplotlib.org/)
- [Seaborn](https://seaborn.pydata.org/)
- [ReportLab](https://www.reportlab.com/) (for PDF generation)

```bash
